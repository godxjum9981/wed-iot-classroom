📚 Smart Classroom – ห้องเรียนอัจฉริยะ

ระบบ Smart Classroom เป็นระบบบริหารจัดการพลังงานและตรวจสอบสภาพแวดล้อมภายในห้องเรียน โดยใช้ IoT (ESP32) เชื่อมต่อกับ Server API และฐานข้อมูล เพื่อเก็บข้อมูลและแสดงผลแบบเรียลไทม์

ระบบสามารถตรวจสอบการใช้งานห้องเรียน ตรวจจับการเคลื่อนไหว และวัดค่าพลังงานไฟฟ้าเพื่อช่วย ลดการใช้พลังงานที่ไม่จำเป็น
